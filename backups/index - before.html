Street Sweeper - Game Prototype

Game Prototype

Street Sweeper is a game about a character named Jack Slate. He is a hamburger flipper who mistakenly gets involved in a war against the criminal underground.

The game prototype was created in Flash CC. It was programmed in ActionScript 3.0. It features dynamic character animations, sound effects, game states, controls, and original artwork.

Trainer Tracker - Mobile App Prototype

Mobile App Prototype

Trainer Tracker is a mobile application that easily logs physical training and nutritional information.

The user simply selects information and adds it. It appears organized in a log page. The user can also view motivational quotes at any time to help boost their performance.

The app was created using Flash CC and ActionScript 3.0.
